# 🎬 ASMR Video Processing Tool

Local web app untuk memproses video ASMR hasil generate AI (Google Flow/Veo).

## Fitur
- ✂️ Crop watermark
- 🔁 Loop video (1 jam / 3 jam / custom)
- 🔊 Inject audio ambient (Brown/Pink/White noise)
- ⬆️ Upscale ke 1080p (Lanczos)
- 🖼️ Extract thumbnail
- 🚀 All-in-One pipeline

## Requirements
- Python 3.10+
- FFmpeg (installed & in PATH)

## Install & Run

### Windows
```bash
pip install -r requirements.txt
start.bat
```

### Linux / CasaOS
```bash
pip install -r requirements.txt
bash start_linux.sh
```

## Akses
Buka browser: `http://localhost:8000`
